## Contexte & Concept
J'ai réalisé un site de streaming à l'aide des outils de développement, avec un backoffice, et j'ai aussi hébergé le site web et réalisé les certifications. Plusieurs fonctionnalités y ont été ajoutés, tels que les profils, la recherche de films, tri par catégorie, ou encore, la possibilité de noter un film.

## Processus  
Le projet s’est construit en plusieurs étapes :  
- Conception du back-office et de la base de données
- Création de l’interface front-end
- Intégration complète front/back 
- Itération finale et reproduction d’un site de streaming existant  

Outils utilisés : Visual Studio Code principalement
